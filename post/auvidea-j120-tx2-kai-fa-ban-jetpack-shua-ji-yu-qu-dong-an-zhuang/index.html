Auvidea J120 TX2开发板 Jetpack刷机与驱动安装

2022-05-05 3 min read # Jetson

一、刷机准备

  1. 下载Nvidia SDK Manager
  • 网址：https://developer.nvidia.com/nvidia-sdk-manager
  • 安装：sudo apt install ./sdkmanager_[version]-[build#]_amd64.deb
  • 打开客户端
    • 普通打开：命令行输入sdkmanager
    • 打开全部版本Jetpack：sdkmanager --archivedversions
  • 登录
    • 推荐下载最新版本Nvidia SDK Manager，旧版本用帐号密码登录可能登不上
  1. 下载Auvidea驱动
  • 驱动网址：https://auvidea.eu/firmware/
    • 根据板子型号、所需安装的Jetpack版本确定下载的驱动文件
  • Auvidea官方刷机文档：https://ardupilot.org/dev/docs/companion-computer-nvidia-tx2.html

二、刷机

  1. 版本
  • jetpack4.6+4.6驱动 不可行，usb驱动刷机不成功
  • jetpack3.3+3.3驱动 可行，对应tensorrt4，版本较低
  • jetpack4.3+4.3驱动 可行，对应tensorrt6
  • jetpack4.4+4.3驱动 可行，对应tensorrt7，最终使用
  1. 打开sdkmanager
  • 第一步，选择产品类型，选择Jetson
  • 第二步，选择安装包，有主机端Host Machine 和目标端Target Hardware，主机端不要选，目标端选择TX2
  • 第三步，选择Jetpack版本
  • 下一步
  • 勾选同意协议，下一步
  • 开始下载，等待下载完成
  1. 弹出SDK Manager is about to flash your Jetson界面
  • 复制驱动文件至目标文件夹，非Auvidea开发板不进行这一步
    • 参考Auvidea官方刷机文档
cp auvidea-kernel-J90-J120-v1.6/auvidea-TX2-Jetpack/dtb/tegra186-quill-p3310-1000-a00-00-base.dtb 64_TX2/Linux_for_Tegra/kernel/dtb
cp auvidea-kernel-J90-J120-v1.6/auvidea-TX2-Jetpack/dtb/tegra186-quill-p3310-1000-c03-00-base.dtb 64_TX2/Linux_for_Tegra/kernel/dtb
cp auvidea-kernel-J90-J120-v1.6/auvidea-TX2-Jetpack/dtb/tegra186-quill-p3310-1000-c03-00-dsi-hdmi-dp.dtb 64_TX2/Linux_for_Tegra/kernel/dtb
  • 选择Manual Setup手动安装
  • 将开发板置于恢复模式
  • 将TX2和主机用网线连接在用一个路由器或者交换机上，用ucb线将主机和TX2连接在一起。让TX2连接上电源适配器，但是断电，按下并松开TX2的POWER键；按下并保持FORCE RECOVERY键；在此期间按下并松开RESET键；等待2秒后松开FORCE RECOVERY键
  • J120开发板只需断电，按下POWER键上电过几秒松开，即置于恢复模式
  • 点击Flash。出错则重新设置恢复模式
  1. 将TX2连接上显示器，待亮屏，此时系统已刷好
  • 选择语言，时区，设置用户名和密码
  • 查看IP
  • 在主机端选择网络连接，输入开发板的IP、用户名、密码
  • 等待安装Target Components
  • 完毕

三、错误

  1. INFO: Flash Jetson TX2 - flash: UnicodeDecodeError: ‘utf-8’ codec can’t decode byte 0xa4 in position 16: invalid start byte
  • 出现在Jetpack4.6版本刷机时
  • 解决方法：在二、3.弹出SDK Manager is about to flash your Jetson界面时不使用sdkmanager进行Flash
  • 使用生成的OS文件夹中的脚本进行Flash
